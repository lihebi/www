Low-Resolution Face Recognition via Sparse Representation of Patches(2013)

Repository on GitHub

Overview

Extract patches with different size from face images, calculate the feature(LBP for example), Than use l1minimization to solve the convex optimization problom and generate

Extract many patches with different sizes from face images, and use these patches to calculate feature, and finally use Adaboost to get a strong classifier

Algorithm

  1. Extract patches from face images
  2. For each patch, calculate feature(LBP for example)
  3. Use l1minimization to solve the convex optimization problem and get weak classifier
  4. Use Adaboost to generate strong classifier from topN weak ones
  5. Use Strong classifier to experiments on the test set on YaleB
